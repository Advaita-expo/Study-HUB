## Troubleshooting

**Database connection error?**
- Ensure MySQL is running
- Check DB credentials in `.env.local`

**Port already in use?**
- Change `SERVER_PORT=5001` in `.env.local`

**Graph not updating?**
- Check browser console (F12)
- Clear cache (Ctrl+Shift+Delete)
